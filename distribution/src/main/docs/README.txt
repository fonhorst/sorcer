Service ORiented Computing EnviRonment (SORCER)

1. Licensing

This binary distribution contains the SORCER platform version ${v.sorcer}. Along with the basic services it is accompanied by the source code of examples that may be used to test and demonstrate the basic features of the SORCER metacomputing platform.

SORCER's core is available as an open source product on the Apache License 2.0. The sources may be downloaded from: https://github.com/sorcersoft/sorcer
Commercial tools training and technical support for the SORCER platform may be obtained from Sorcersoft.com S.A. (http://www.sorcersoft.com).

This distribution contains third party software. Among it is the Berkley DB (Sleepycat) distributed under the Sleepycat license. Sleepycat's source code is available at: http://www.oracle.com/us/products/database/berkeley-db/
You can find licenses of all libraries distributed along SORCER in the SORCER_HOME/docs/licenses folder.
